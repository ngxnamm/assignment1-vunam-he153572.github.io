Web design


A web designer is someone who converts ideas or stories into visually appealing designs. By using layout to build user experience across the entire website. A web designer like an architect will create a plan of your house before starting to build it. They will model the website layout before a developer and start web development.



Interaction design


Interaction design is the stage where app developers divide the interaction and use tangible elements like animation. From there, the tool will enhance the web-app experience to a new level by observing how users communicate and guide their reactions and behaviors.

Graphic design


Graphic Design is an art product that harmoniously combines elements such as typography and images to convey a certain message to the viewer. Graphic Design's products are available everywhere; It can be magazines, movie posters or billboards, etc. Each product is presented with its own style but has the sole purpose of directing viewers to a specific message. Graphic Designer is a person who does graphic design work. They use specialized software to create different products.
